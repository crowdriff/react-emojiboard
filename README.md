# react-emojiboard
## Installation
```
npm install react-emojiboard
```

## Supported Emojis
Full Emoji List: http://unicode.org/emoji/charts/full-emoji-list.html

## Development
```
git clone git@github.com:crowdriff/react-emojiboard
cd react-emojiboard
npm i
npm start
```

...and point your browser to `http://localhost:8080`.

## Publish
```
npm run build
npm publish
```
